# BenchVault v1.0.2

This prerelease patch adds repeatable CI and tag-based release packaging.

## Changes

- Added GitHub Actions CI for format, tests, analyzer, safety scans, seeder
  dry-run, and macOS build.
- Added a tag-triggered GitHub Actions release workflow that builds the unsigned
  macOS prerelease zip and uploads it to the matching GitHub release.
- Added `scripts/package_macos_release.sh` for local and CI packaging.
- Updated release documentation and README with the new automation path.

## Verification

- `flutter test`
- `flutter analyze`
- `scripts/release_smoke_check.sh`
- `scripts/package_macos_release.sh 1.0.2`
- secret and absolute-path scans for public files and the generated PDF

## Important Limits

- The macOS app artifact remains unsigned and not notarized.
- The release workflow packages prerelease builds; signed production
  distribution still requires Apple Developer signing/notarization credentials.
- Windows and iPad remain scaffolded but unvalidated as release platforms.

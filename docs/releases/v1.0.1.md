# BenchVault v1.0.1

This prerelease patch adds macOS Keychain-backed secret storage for the app path
and keeps the existing ignored local-file fallback for tests, tools, and
non-macOS platforms.

## Changes

- Added a secure secret-store abstraction.
- Added macOS Keychain storage for LabArchives GOV access ID/access key and the
  OpenAI API key when the macOS app is launched normally.
- Kept non-secret setup metadata, UID records, notebook indexes, schedules, and
  integrity ledgers in ignored local files.
- Migrates legacy local OpenAI/LabArchives secret files to Keychain when the
  macOS app reads them.
- Updated quickstart, limitations, release checklist, strategic plan, and
  developer reference to describe the new storage behavior.

## Important Limits

- The macOS app artifact is still not signed or notarized.
- Windows Credential Manager and iPad Keychain support are not implemented.
- Windows and iPad remain scaffolded but unvalidated as release platforms.
